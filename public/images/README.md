# Images Directory

This directory contains all the images used in the website. Here's what you should add:

## Required Images

1. **avatar.jpg** - Your profile photo (recommended size: 300x300px)
2. **microservice-architecture.png** - For the blog post (recommended size: 800x600px)
3. **street-photography.jpg** - For photography page (recommended size: 800x600px)
4. **landscape-photography.jpg** - For photography page (recommended size: 800x600px)
5. **portrait-photography.jpg** - For photography page (recommended size: 800x600px)

## Image Guidelines

- Use JPG format for photos
- Use PNG format for diagrams and screenshots
- Optimize images for web (compress them)
- Keep file sizes under 500KB for better loading performance
- Use descriptive filenames

## Adding New Images

1. Place your image files in this directory
2. Reference them in your content using the path `/images/filename.ext`
3. Make sure the filenames match exactly what's referenced in the content files

## Example Usage

In your markdown files, reference images like this:
```markdown
![Description](/images/filename.jpg)
```
