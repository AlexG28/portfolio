# Projects

Here's a collection of projects I've worked on, showcasing my skills in software development, system design, and problem-solving.

## Featured Projects

### Microservice API Gateway
**Technologies**: Go, Docker, gRPC, PostgreSQL

A high-performance API gateway built with Go that handles authentication, rate limiting, and request routing for a microservice architecture. Features include:

- JWT-based authentication
- Rate limiting with Redis
- Request/response logging
- Health check endpoints
- Docker containerization

*[View on GitHub](https://github.com/AlexG28/api-gateway)*

### Task Management System
**Technologies**: Python, FastAPI, PostgreSQL, React

A full-stack task management application with real-time updates and collaborative features. Built with FastAPI for the backend and React for the frontend.

- User authentication and authorization
- Real-time task updates
- File upload functionality
- RESTful API design
- Responsive web interface

*[View on GitHub](https://github.com/AlexG28/task-manager)*

### Data Pipeline Orchestrator
**Technologies**: Python, Apache Airflow, Docker, AWS

An ETL pipeline orchestrator that processes large datasets and provides data analytics capabilities. Features include:

- Automated data extraction from multiple sources
- Data transformation and cleaning
- Scheduled job execution
- Error handling and retry mechanisms
- Monitoring and alerting

*[View on GitHub](https://github.com/AlexG28/data-pipeline)*
